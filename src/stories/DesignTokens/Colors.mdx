## Uso em Código

```css
/* CSS Variables */
.elemento {
  background-color: hsl(var(--primary));
  color: hsl(var(--primary-foreground));
}

/* Tailwind Classes */
<div className="bg-primary text-primary-foreground" />
<span className="text-status-otimo" />
```

```tsx
// TypeScript
import { colors } from '@/lib/design-tokens';

// Acessar valores
const primaryHsl = colors.primary.hsl; // "214 100% 50%"
const otimoHex = colors.status.otimo.hex; // "#3C8DBC"
```
