# Espaçamento

O sistema de espaçamento segue a escala do Tailwind CSS, baseada em múltiplos de 4px.

## Escala Base

<div style={{ marginTop: '24px', display: 'flex', flexDirection: 'column', gap: '8px' }}>
  {[
    { name: '0.5', value: '2px', rem: '0.125rem' },
    { name: '1', value: '4px', rem: '0.25rem' },
    { name: '1.5', value: '6px', rem: '0.375rem' },
    { name: '2', value: '8px', rem: '0.5rem' },
    { name: '3', value: '12px', rem: '0.75rem' },
    { name: '4', value: '16px', rem: '1rem' },
    { name: '5', value: '20px', rem: '1.25rem' },
    { name: '6', value: '24px', rem: '1.5rem' },
    { name: '8', value: '32px', rem: '2rem' },
    { name: '10', value: '40px', rem: '2.5rem' },
    { name: '12', value: '48px', rem: '3rem' },
    { name: '16', value: '64px', rem: '4rem' },
  ].map(({ name, value, rem }) => (
    <div key={name} style={{ display: 'flex', alignItems: 'center', gap: '16px' }}>
      <code style={{ width: '50px' }}>{name}</code>
      <div style={{ width: value, height: '24px', backgroundColor: 'hsl(214, 100%, 50%)', borderRadius: '4px' }} />
      <span style={{ color: '#64748B', fontSize: '14px' }}>{value} ({rem})</span>
    </div>
  ))}
</div>

## Uso Semântico

| Token | Valor | Uso Comum |
|-------|-------|-----------|
| `gap-1` | 4px | Espaço entre ícone e texto |
| `gap-2` | 8px | Espaço entre elementos inline |
| `gap-4` | 16px | Espaço entre campos de formulário |
| `gap-6` | 24px | Espaço entre seções |
| `p-3` | 12px | Padding de células de tabela |
| `p-4` | 16px | Padding de cards pequenos |
| `p-5` | 20px | Padding de cards médios |
| `p-6` | 24px | Padding de cards grandes |
| `my-4` | 16px | Margem vertical entre componentes |
| `my-6` | 24px | Margem vertical entre seções |

## Uso em Código

```tsx
// Tailwind Classes
<div className="p-4 gap-2">...</div>
<div className="mt-6 mb-4">...</div>
<div className="space-y-4">...</div>

// Design Tokens
import { spacing, semanticSpacing } from '@/lib/design-tokens';

spacing[4]; // "1rem"
semanticSpacing.cardPadding; // "1.25rem" (20px)
semanticSpacing.gapMd; // "1rem" (16px)
```
