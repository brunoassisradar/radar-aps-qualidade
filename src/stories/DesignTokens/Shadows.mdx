## Transições

| Classe | Duração | Uso |
|--------|---------|-----|
| `duration-75` | 75ms | Micro-interações |
| `duration-150` | 150ms | Hovers rápidos |
| `duration-200` | 200ms | Padrão (botões, links) |
| `duration-300` | 300ms | Transições de estado |
| `duration-500` | 500ms | Animações suaves |

## Uso em Código

```tsx
// Sombras
<div className="shadow-sm">Sutil</div>
<div className="shadow-lg">Destaque</div>

// Border Radius
<div className="rounded-lg">Card</div>
<div className="rounded-full">Avatar</div>

// Transições
<button className="transition-colors duration-200 hover:bg-primary/90">
  Botão
</button>
```

```tsx
// Design Tokens
import { shadows, borderRadius, transitions } from '@/lib/design-tokens';

shadows.lg; // "0 10px 15px -3px rgb(0 0 0 / 0.1), ..."
borderRadius.lg; // "0.5rem"
transitions.duration.normal; // "200ms"
```
